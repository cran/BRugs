General properties [top]

The commands change the appearance of text in documents. They either act on the highlighted stretch of text or affect the appearance of subsiquently entered text.


Weight [top]

The commands in this group set the weight of the highlighted text. The weight can either be Regular, Bold, Italic or Underline. If the entire highlighted text is of one single weight a tick mark will appear against the relavent weight.


Fixed Size [top]

The commands in this group change the size of the highlighted text to a predefined font size. Sizes available are 8, 9, 10, 12, 16, 20 and 24pts. If the entire highlighted text is of one single font size a tick mark will appear against the relavent size.


Size... [top]

This command opens a non modal dialog box which allows the user to choose the font size for the highlighted text.

         
         
         
Fixed Color [top]

The commands in this group change the colur of the highlighted text to a predefined colour. Colours available are Default Color, Black, Red, Green, and Blue. If the entire highlighted text is of one single colour a tick mark will appear against this colour.


Color... [top]

This command opens a modal dialog box which allows the user to choose the colour for the highlighted text.

         
         
         
Default Font [top]

This command changes the highlighted text to be in the default font. If the entire highlighted text is in the defaukt font a tick mark will appear against this option.


Font... [top]

This command opens a modal dialog box which allows the user to choose the font for the highlighted text. The colour of the text can also be choosen from a restricted range of options. This command is useful for entering greek characters into a document.

         
         
         
         
Typeface... [top]

This command is a simplified version of the Font... command without the ability to change the colour of the text.
